# Covid--19-Dashboard
COVID-19 Analytics Dashboard | Power BI

Designed and developed an interactive Power BI dashboard to analyze COVID-19 trends using advanced data visualization techniques. Implemented data cleaning with Power Query, created DAX measures, built data models, and developed dynamic reports with filters and drill-down capabilities to deliver actionable insights.
